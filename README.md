# Intelligent Document Analyzer — Azure

Application web d'analyse de documents intelligente déployée sur Microsoft Azure.

## Stack
- **Backend** : Python / FastAPI
- **Frontend** : React / Azure Static Web Apps
- **Infrastructure** : Azure Bicep (IaC)
- **CI/CD** : GitHub Actions
- **IA** : Azure Document Intelligence + Azure OpenAI GPT-4o
